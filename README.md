# Baudekoration Nonnenmacher

Kompletter statischer Relaunch der Website `baudekoration-nonnenmacher.de`.

## Inhalt

- `index.html` - vollständige One-Page-Website
- `styles.css` - responsive Gestaltung, Layout, Animationen
- `script.js` - Navigation, Galerie, Filter, Dialoge und E-Mail-Anfrage
- `assets/` - gesicherte Bilder und Logos der bestehenden Website
- `robots.txt` und `sitemap.xml` - technische SEO-Grundlage

## Veröffentlichung

Die Seite ist für GitHub Pages vorbereitet und kann direkt aus dem `main`-Branch veröffentlicht werden.
